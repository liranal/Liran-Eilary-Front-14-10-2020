  <h2>Installation Guide</h2>
  <h4>
  1. Clone Client<br>
  2. run npm install
  on the client directory<br>
  <br>
  3. Clone Server<br>
  4. run  npm install
  on the server directory<br>
  <br>
  6. run On server directory `npm start`<br>
  7. run on client directory `npm start`<br>
  </h4>
</p>
